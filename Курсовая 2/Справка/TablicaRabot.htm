Закладка 2 - Таблица работ

На данной закладке расположена таблица с семью столбцами и двумя кнопками. Данная закладка предназначена для хранения данных, а также для дальнейшей обработки этих данных на следующей закладке - графике. Закладка показана ниже.

Рассмотрим каждый элемент закладки

  1. Таблица созданных работ - таблица, куда записываются данные, введенные пользователем на предыдущей закладке. Таблица имеет 7 столбцов, которые соответсвуют полям из предыдущей закладки. Редактировать данные самостоятельно в таблице нельзя.
  2. Кнопка "Записать таблицу в CSV файл" - кнопка, предназначенная для записи данных из таблицы в CSV файл.
  3. Кнопка "Записать в таблицу из CSV файла" - кнопка, предназначенная для записи данных из CSV файла в таблицу.

Вводить данные в таблицу можно двумя способами:

  1. Вводить данные на первой закладке и с помощью кнопки "Записать" добавлять эти данные в таблицу.
  2. Использовать кнопку "Записать в таблицу из CSV файла" для заполнения данными из CSV файла.
